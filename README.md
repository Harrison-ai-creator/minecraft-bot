# Minecraft-bot
Этот скрипт запускает автоматических ботов для майнкрафта, они облегчат вашу игру.
## Их возможности
Афк бот:
* Телепортироваться по команде
* Автоматически насыщать себя
* Прогружать чанки
  
Бот шахтер:
* Телепортироваться по команде
* Автоматически насыщать себя
* Искать и добывать руды\блоки
* Складывать вещи в сундуки

## Требования для установки
Перед запуском установите программу node.js >= 18 из [node.js](https://nodejs.org).

## Установка
1. **Скачайте код** (через кнопку Code -> Download ZIP или команду 'git clone').

2. Откройте папку с ботом, нажмите правой кнопкой мыши по пустому месту и выберите **"Открыть в терминале"** (или командной строке).

3. **Установить все нужные библиотеки.** Для этого введите команду(может занять 1-2 минуты):
```bash
npm install
```
4. **Редактирование бота.** Откройте файл bot.js любым текстовым редактором, поменяйте никнеймы ботов или добавьте их, а затем напишите в терминале: **node bot.js**

## Запуск бота

После установки всех библиотек, в открытом терминале введите команду:
```bash
node bot.js
```

## Управление несколькими ботами
Скрипт поддерживает запуск нескольких ботов одновремено. Чтобы добавить или изменить ботов, откройте файл скрипта.
### 1. Как добавить новых ботов в коде
Функции createMyBot и createDiggerBot принимают **Имя бота**, **Порядковый номер (ID)**, где **Хост**, **Порт** и **Версия** скрипт спрашивает сам и их не нужно менять.

```javascript
//createMyBot('Имя_бота', Номер, хост, порт, версия)
createMyBot('Bot_2', 2, host, portinput, version) //Бот №1
createDiggerBot('BotDigger', 3, host, portinput, version) //Бот №2
```

### Команды:
1. tp[Номер бота] [x] [y] [z]. Пример: tp1 100 60 -80.
2. mine[Номер бота] [Название блока на англиском]. Пример: mine3 diamond_ore.
3. leave[Номер бота]. Пример: l3.
4. !status - Показывает прочность инструмента и что лежить в руке.
5. !inv - Показывает что лежить в инвентаре.
6. !stop - Останавливает бота(ов).
7. !sethome - Устанавливает точку дома.
8. !home - Отправляет бота(ов) на точку дома.
9. !throw - Бот(ы) складывают вещи из инвенторя.
