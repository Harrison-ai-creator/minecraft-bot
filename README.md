# Minecraft-bot
Этот скрипт запускает автоматических ботов для майнкрафта, они облегчат вашу игру.
## Их возможности
Афк бот:
* Телепортироваться по команде
* Автоматически насыщать себя
* Прогружать чанки
  
Бот шахтер:
* Телепортироваться по команде
* Автоматически насыщать себя
* Искать и добывать руды\блоки
* Складывать вещи в сундуки

## Требования для установки
Перед запуском установите программу node.js >= 18 из [node.js](https://nodejs.org).

## Установка
1. **Скачайте код** (через кнопку Code -> Download ZIP или команду 'git clone').

2. Откройте папку с ботом, нажмите правой кнопкой мыши по пустому месту и выберите **"Открыть в терминале"** (или командной строке).

3. **Установить все нужные библиотеки.** Для этого введите команду:
```bash
npm install
```
4. **Запустите бота.** Откройте файл bot.js любым текстовым редактором, поменяйте никнеймы ботов или добавьте их, а затем напишите в терминале: **node bot.js**
